
Description
-----------
This module is a simple mortgage calculator. Currently it just calculates the monthly payment amount, but in the future you will be able to see the amortization table bellow that.


Installation
------------
1) Place this module directory in your modules folder (this will usually be
   "sites/all/modules/").

2) Enable the module within your Drupal site at Administer -> Site Building ->
   Modules (admin/build/modules).

Usage
-----
1. Download and enable the Mortgage Calculator module
2. Go to  admin/structure/block to enable the "Mortgage Calculator" block.

Dependencies
--------
1. The block modules needs to be installed.
2. JQuery version 1.6 or higher is required for this module to work. JQuery Update module
needs to be installed.
